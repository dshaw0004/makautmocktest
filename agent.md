# AGENTS.MD

This file provides instructions for AI agents working with this codebase.

## Project Overview

This project is a mock test website for students of the MAKAUT (Maulana Abul Kalam Azad University of Technology) to practice for MCQ-based exams. It is built with React, Vite, and Shadcn UI, and uses the Google Gemini API to generate question sets.

## Setup and Installation

To get started with the project, you need to have Node.js and npm installed. Then, follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

## How to Run the Project

To run the project locally for development, use the following command:

```bash
npm run dev
```

This will start the development server, and you can view the application in your browser at `http://localhost:5173` (the port may vary).

## Tech Stack

-   **Framework:** React
-   **Build Tool:** Vite
-   **UI:** Shadcn UI
-   **State Management:** Zustand
-   **Routing:** React Router
-   **Styling:** Tailwind CSS

## Code Structure

The source code is located in the `src` directory and is organized as follows:

-   `src/components`: Contains reusable UI components.
-   `src/pages`: Contains the main pages of the application.
-   `src/hooks`: Contains custom React hooks.
-   `src/store`: Contains the Zustand store for state management.
-   `src/lib`: Contains utility functions.
-   `src/types`: Contains TypeScript type definitions.

## UI Design

The UI design files for this project were created using [Stitch by Google](https://stitch.withgoogle.com/projects/5993827729178666057?pli=1) and are located in the `/ui-design` directory.
